# knowledge-tools

Tools supporting a personal space industry knowledge base built on the WikiAI flat-file system.

## Tool Categories

- **automation/** — Scraping, raw/ ingestion, scheduled jobs
- **health/** — Wiki checks, contradiction detection, gap analysis
- **pipelines/** — Data processing, format conversion, enrichment
- **distribution/** — Telegram, iMessage, email, webhook outputs
- **federation/** — Syncing and collaborating with other knowledge bases
- **lib/** — Shared utilities used across all tool categories
